# NAC工程架构解读

## 工程定位

**NAC (NexusAgent-Cluster)** 是一个多Agent集群编排系统，具备DAG并行调度和Agent自进化能力。

## 核心架构

```
┌─────────────────────────────────────────┐
│           用户输入层                    │
│    (CLI Chat / REST API)               │
└──────────────┬──────────────────────────┘
               │
┌──────────────▼──────────────────────────┐
│         Orchestrator (主编排器)         │
│  ┌──────────────────────────────────┐  │
│  │ 1. IntentParser  → 解析意图      │  │
│  │ 2. DAGBuilder    → 构建依赖图    │  │
│  │ 3. Scheduler     → 并行调度      │  │
│  │ 4. AgentRouter   → 智能路由      │  │
│  └──────────────────────────────────┘  │
└──────────────┬──────────────────────────┘
               │
     ┌─────────┼─────────┬─────────┐
     │         │         │         │
┌────▼──┐ ┌───▼──┐ ┌──▼───┐ ┌─▼────┐
│ Code  │ │ Data │ │Agent │ │Skill │
│Agent  │ │Agent │ │Route │ │Mgr  │
└───────┘ └──────┘ └──────┘ └──────┘
                             │
                    ┌────────┴────────┐
                    │ 8个内置Skills   │
                    │ ⭐ skill-creator│
                    └─────────────────┘
```

## 核心特性

1. **DAG并行调度**: 自动识别可并行任务
2. **智能Agent路由**: 基于语义匹配选择Agent
3. **可扩展Skills系统**: 插件式架构
4. **黑板模式+事件总线**: 共享状态和通信
5. **Agent自进化**: 基于反馈持续优化

## Skills系统 (8个)

1. code-generation - 代码生成
2. file-ops (v1.1.0) - 文件操作 ⭐安全加固
3. terminal-exec - 终端命令
4. code-review - 代码审查
5. data-analysis - 数据分析
6. docx-processing - Word文档
7. web-search - 网络搜索
8. skill-creator - 创建Skill ⭐NEW
