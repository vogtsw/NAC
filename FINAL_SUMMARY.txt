========================================================================

工程定位: 多Agent集群编排系统

核心组件:
  • Orchestrator - 主编排器
  • IntentParser - 意图解析器
  • DAGBuilder - DAG构建器
  • Scheduler - 任务调度器
  • AgentRouter - Agent路由器
  • SkillManager - Skill管理器 (8个Skills)

8个内置Skills:
  1. code-generation (代码生成)
  2. file-ops v1.1.0 (文件操作) ⭐ 已加固安全

  6. docx-processing (Word文档)
  7. web-search (网络搜索)
  8. skill-creator (创建Skill) ⭐ NEW

核心特性:

  • 可扩展Skills - 插件式架构

  • Agent自进化 - 持续优化

========================================================================
🎯 skill-creator 使用方法
========================================================================

方法1: CLI交互模式 (推荐)
----------------------------------------
pnpm cli chat

You> 创建一个新skill：
   - skill名称: email-sender
   - 描述: 使用SMTP协议发送邮件
   - 类别: automation
   - 必需参数: to, subject, body
   - 可选参数: cc, bcc

方法2: 编程方式
----------------------------------------
const skillManager = await getSkillManager();
const creatorSkill = skillManager.getSkill('skill-creator');

const result = await creatorSkill.execute(
  { logger: console },
  {
    skillName: 'email-sender',
    description: '发送邮件功能',
    category: 'automation',
    parameters: {
      required: ['to', 'subject', 'body'],
      optional: ['cc', 'bcc']
    }
  }
);
