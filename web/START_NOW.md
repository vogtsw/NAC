## 🔍 如果还有问题

### 检查清单
```bash
# 1. 确认服务启动
pnpm web
# 应该看到: "Server listening at http://127.0.0.1:3000"

# 2. 确认端口可用
curl http://localhost:3000/health
# 应该返回: {"status":"ok",...}

# 3. 查看日志
# 终端会显示详细的初始化和请求日志
```

### 常见错误

**错误1: 端口被占用**
```
Error: listen EADDRINUSE: address already in use :::3000
解决: 关闭占用3000端口的程序
```

**错误2: API密钥未配置**
```
Error: API key not configured
解决: 检查.env文件中的ZHIPU_API_KEY
```

**错误3: 依赖缺失**
```
Error: Cannot find module '@fastify/static'
解决: pnpm install
```
