## 🔍 问题排查

### 如果服务无法启动

**症状：** 运行`pnpm web`后立即退出

**检查：**
```bash
# 1. 检查端口是否被占用
netstat -ano | findstr :3000

# 2. 检查.env文件
cat .env | grep ZHIPU_API_KEY

# 3. 检查依赖安装
pnpm list fastify @fastify/static
```

**解决：**
- 如果端口被占用：关闭占用进程或修改端口
- 如果API密钥未配置：编辑.env文件
- 如果依赖缺失：运行`pnpm install`

### 如果浏览器显示"无法访问"

**检查：**
1. 服务是否真的启动了？（查看终端输出）
2. 防火墙是否阻止了连接
3. 浏览器地址是否正确（http://localhost:3000）

### 如果页面显示空白

**检查：**
1. 打开浏览器开发者工具（F12）
2. 查看Console是否有错误
3. 查看Network标签页，检查资源加载情况

## 💡 使用技巧

### 1. 快捷键
- `Ctrl + Enter` - 在输入框中直接执行任务（需要自行添加）
- `Esc` - 清空输入

### 2. 查看详细日志
- 勾选"自动滚动"复选框
- 日志会自动滚动到最新

### 3. 导出结果
- 点击"导出"按钮
- 结果保存为`.txt`文件
- 包含完整的时间戳和内容

### 4. 多任务测试
- 不需要停止服务
- 可以连续提交多个任务
- 系统会自动管理会话
